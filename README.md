# S.I Métodos de aprendizaje supervisado

Proyecto de Inteligencia Artificial - Implementar un modelo de aprendizaje automático supervisado que permita predecir el tiempo de viaje utilizando datos históricos simulados del sistema de transporte masivo.

## Descripción

Este proyecto tiene como objetivo aplicar técnicas de Aprendizaje Supervisado para predecir el tiempo estimado de viaje en un sistema de transporte masivo similar a TransMilenio.

Se desarrolló un modelo de Regresión Lineal utilizando Python y la biblioteca Scikit-Learn, capaz de estimar la duración de un recorrido a partir de variables como la hora del viaje, la cantidad de pasajeros y la distancia recorrida.

## Tecnologías utilizadas
Python 3.14
Pandas
Scikit-Learn
NumPy
